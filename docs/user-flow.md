# ユーザーフロー

## 概要

このドキュメントは、AI通知アプリのユーザーフローをまとめたものです。ユーザーがアプリをインストールしてから、通知を受け取り、活用するまでの流れを記述しています。主な目的は、学習やモチベーション維持のためのAI生成通知を、指定した時間に受動的に受け取ることです。

## 主要なフロー

1. **アプリのインストールと初期設定**
2. **通知ルールの作成**
3. **通知の受信と閲覧**
4. **通知履歴の確認**
5. **設定の変更と管理**

## 詳細なステップ

### 1. アプリのインストールと初期設定

- **ステップ1.1**: ユーザーがApp StoreまたはGoogle Playからアプリをダウンロード・インストールする。
- **ステップ1.2**: アプリを初回起動し、利用規約とプライバシーポリシーに同意する。
- **ステップ1.3**: プッシュ通知の許可を求めるダイアログが表示され、ユーザーが許可する。
- **ステップ1.4**: 基本的なプロフィール設定（オプション：名前、目標など）を入力する。

### 2. 通知ルールの作成

- **ステップ2.1**: ホーム画面から「通知ルール作成」ボタンをタップする。
- **ステップ2.2**: 通知したい時刻を選択する（例: 毎日15:00）。
- **ステップ2.3**: テーマを選択または入力する（例: やる気、睡眠、学習）。
- **ステップ2.4**: 通知の頻度を設定する（毎日、平日など）。
- **ステップ2.5**: ルールを保存し、確認画面で有効化する。

### 3. 通知の受信と閲覧

- **ステップ3.1**: 指定した時刻になると、プッシュ通知が届く。
- **ステップ3.2**: 通知をタップすると、アプリが開き、通知詳細画面が表示される。
- **ステップ3.3**: 通知内容を確認する：
  - 短い通知文（例: 「継続は力なり」）
  - 元となる名言や知識
  - 背景や意味の解説
  - 今の自分に活かすための行動提案
- **ステップ3.4**: 必要に応じて、関連リンクや出典を確認する。
- **ステップ3.5**: 通知を気に入った場合、お気に入りに追加する（オプション）。

### 4. 通知履歴の確認

- **ステップ4.1**: ホーム画面またはメニューから「履歴」タブに移動する。
- **ステップ4.2**: 過去の通知をリスト形式で閲覧する。
- **ステップ4.3**: 特定の通知をタップして詳細を確認する。
- **ステップ4.4**: 検索やフィルタ機能を使って、テーマや日付で絞り込む。

### 5. 設定の変更と管理

- **ステップ5.1**: 設定画面にアクセスする。
- **ステップ5.2**: 既存の通知ルールを編集または削除する。
- **ステップ5.3**: プッシュ通知のオン/オフを切り替える。
- **ステップ5.4**: アカウント設定を変更する（例: プロフィール更新）。
- **ステップ5.5**: アプリのバージョン情報やヘルプを確認する。

## 補足

- すべての通知はAI（OpenAI API）によって動的に生成され、飽きにくい内容を提供します。
- ユーザーは通知をきっかけに、学習や行動改善につなげることができます。
- 将来的には、パーソナライズ機能（ユーザーの過去の反応に基づく調整）やサブスクリプション機能を追加予定です。
