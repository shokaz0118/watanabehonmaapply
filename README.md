# AI通知アプリ（仮）

## 概要

指定した時間に、AIが学びや気づき、励ましを届けるモバイルアプリです。
ユーザーは「通知したい時刻」と「テーマ」を設定するだけで、日常の中で受動的に知識や刺激を受け取ることができます。

## 主な機能

* 通知ルールの作成（通知が欲しい時刻・テーマ設定）
* 定時でのプッシュ通知

* 本アプリでは、通知ごとに以下のような内容が生成されます。
短く読みやすい通知文<br>
元となる名言や知識<br>
背景や意味をわかりやすく解説<br>
今の自分に活かすための行動提案<br>
例：
「継続は力なり」<br>
この言葉は、小さな努力でも積み重ねることで大きな成果につながることを示しています。
今日1つでもいいので、小さな行動を積み重ねてみましょう。例えばこんなことから始めてみたらどう？
* 通知詳細の表示（解説・出典・リンク）
* 通知履歴の閲覧

## 利用イメージ

* 毎日15時にやる気が出る名言を受け取る
* 毎日23時に睡眠に関する雑学を知る
* 朝に前向きな一言を受け取る

## 特徴

* 固定文ではなく、AIが内容を生成するため飽きにくい
* 通知をきっかけに「理解」や「行動」につなげられる
* 学習とセルフマネジメントの両方に活用できる

## 想定ユーザー

* 学習を習慣化したい人
* モチベーションを維持したい人
* 忙しくても効率よく知識を得たい人

## 技術スタック（予定）

* フロントエンド: Expo / React Native / TypeScript
* バックエンド: Supabase
* AI: OpenAI API
* 通知: Expo Push Notifications

## 今後の展望

## 開発目的

日常の中で「自分の欲しい情報を受動的に得られる体験」を提供することを目的としています。
ユーザーが無理なく継続できる仕組みを、AIと通知を組み合わせて実現します。
